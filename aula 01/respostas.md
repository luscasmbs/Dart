# Aula 01 - Respostas

## 1. Desenvolvimento Mobile

Desenvolvimento mobile é a criação de aplicativos para celulares e tablets. Esses aplicativos podem ajudar em comunicação, estudos, compras, transporte e diversão. Dois exemplos do meu dia a dia são WhatsApp e YouTube.

## 2. Aplicações Mobile

Um aplicativo mobile é diferente de um sistema de computador porque ele é feito para telas menores, usa toque na tela e pode ser usado em qualquer lugar. Ele também pode usar recursos do aparelho, como câmera, GPS, microfone e notificações.

## 3. Android e iOS

Android e iOS são sistemas operacionais para celulares. O Android é usado em aparelhos de várias marcas, como Samsung, Motorola e Xiaomi. O iOS é usado nos aparelhos da Apple, como o iPhone.

## 4. Aplicativo Mobile, Web e Desktop

Aplicativo mobile é feito para celular ou tablet, como o WhatsApp. Sistema web funciona pelo navegador, como o Google Classroom. Sistema desktop é instalado no computador, como o Word ou o Paint.

## 5. Aplicações Nativas

Uma aplicação nativa é feita especialmente para um sistema, como Android ou iOS. Uma vantagem é que ela pode ter bom desempenho e usar melhor os recursos do celular. Uma desvantagem é que pode ser necessário criar versões diferentes para cada sistema.

## 6. Aplicações Híbridas

Uma aplicação híbrida mistura características de sites com aplicativos mobile. Ela pode usar tecnologias parecidas com as da web, mas funcionar dentro de um aplicativo instalado no celular.

## 7. Aplicações Multiplataforma

Aplicação multiplataforma é aquela que pode funcionar em mais de um sistema usando uma base de código parecida. O Flutter é interessante para equipes pequenas porque permite criar aplicativo para Android e iOS usando Dart.

## 8. Flutter

Flutter é um framework usado para criar aplicativos mobile. Ele ajuda a montar telas, botões, textos, menus e outras partes visuais do aplicativo.

## 9. Widgets no Flutter

Widget é uma parte da interface de um aplicativo Flutter. Quase tudo no Flutter é feito com widgets. Exemplos: Text, Container, Row e Column.

## 10. Dart e Flutter

Dart é a linguagem de programação. Flutter é o framework usado para criar aplicativos. Ou seja, usamos a linguagem Dart para programar dentro do Flutter.
