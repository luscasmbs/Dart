# Aula 03 - Estrutura de Dados 1

Esta pasta contém respostas e exemplos simples em Dart sobre listas.

Nos slides, o exercício pede respostas sobre como adicionar elementos em uma lista usando a ideia de `push`. Em Dart, o método usado em listas é `add()`, e para vários itens é `addAll()`.

- `respostas.md`: respostas curtas das 5 perguntas.
- `exercicio_01.dart` até `exercicio_05.dart`: exemplos práticos em Dart.

Para executar um código:

```bash
dart run exercicio_01.dart
```
